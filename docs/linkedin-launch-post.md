# LinkedIn Launch Post

I built ClickCron: a tiny TypeScript CLI for turning browser clicks into scheduled Playwright checks.

The idea is simple:

- record a browser flow once
- run it locally or in CI
- schedule it with cron-style aliases
- keep logs, screenshots, and JSON results for every run

Useful for price checks, form smoke tests, screenshot monitors, job-board checks, internal dashboard checks, and all the tiny browser tasks that are too important to keep doing manually.
