# ClickCron

ClickCron is a TypeScript CLI scaffold for building scheduled browser automation and monitoring workflows.

## Requirements

- Node.js 20+
- npm 10+

## Quick start

```bash
npm install
npm run typecheck
npm run lint
npm test
npm run build
```

## Scripts

- `npm run dev` - run the CLI entrypoint in development mode.
- `npm run build` - compile and bundle the CLI with tsup.
- `npm run test` - run unit tests with Vitest.
- `npm run lint` - lint all files with ESLint.
- `npm run format` - check formatting with Prettier.
- `npm run typecheck` - run strict TypeScript checks.
